This package helps you write source code in your articles
and make sure it looks nice. Install it from CTAN and then
use like this (pay attention to the \ff{\char`\\ff} command
and the \ff{ffcode} environment):

\begin{multicols}{2}
\scriptsize
\raggedcolumns
\begin{verbatim}
\documentclass{article}
\usepackage{ffcode}
\begin{document}
The function \tt{fibo()} is recursive:
\begin{ffcode}
void fibo(int n) {
  if (n < 2) {
    return n; |$\label{ln:ret}$|
  }
  return fibo(n - 1) + fibo(n - 2);
}
\end{ffcode}
\end{document}
The line~\ref{ln:ret} terminates recursion.
\end{verbatim}

\columnbreak

This C++ function \ff{fibo()} is recursive:

\begin{ffcode}
void fibo(int n) {
  if (n < 2) { |$\label{ln:ret}$|
    return n;
  }
  return fibo(n - 1) + fibo(n - 2);
}
\ end{ffcode}
\end{ffcode}

The line~\ref{ln:ret} terminates recursion.
\end{multicols}

More details about this package you can find
in the \ff{yegor256/ffcode} GitHub repository.
